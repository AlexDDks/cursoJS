# Curso de JavaScript

Este repositorio contiene materiales y ejercicios para mi curso personal de JavaScript. El curso abarca desde conceptos básicos hasta aspectos avanzados del lenguaje.

## Contenido

1. **Introducción a JavaScript**
   - Historia de JavaScript
   - Configuración del entorno de desarrollo

2. **Fundamentos de JavaScript**
   - Tipos de datos primitivos y no primitivos
   - Variables (`var`, `let`, `const`)
   - Operadores
   - Estructuras de control (condicionales, bucles)

3. **Funciones**
   - Declaración de funciones
   - Expresiones de funciones
   - Funciones flecha
   - Cierres (Closures)

4. **Objetos y Arrays**
   - Trabajando con objetos
   - Métodos de objetos
   - Arrays y sus métodos
   - Destructuración

5. **Programación Asíncrona**
   - Callbacks
   - Promesas
   - Async/Await

6. **Avanzado**
   - Prototipos y herencia
   - ES6+ características
   - Manejo de errores
   - Patrones de diseño
